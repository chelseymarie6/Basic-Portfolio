About Me

Hello there! My name is Chelsey Dolan, and I am a student web developer at UCSD Extension. The program is rigorous and fast-paced, covering both the theory and real-world application of full stack web development. Throughout the program I will be building and deploying all types of projects including: front-end websites using HTML/CSS/JavaScript, a browser-based role playing game, a front-end application to solve a real-world problem, a full stack web application with a robust front-end/back-end and scalable database, and several other projects using the latest web technologies.
